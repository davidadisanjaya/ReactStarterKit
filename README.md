###Getting Started###

####Clone This Repo?#####
Clone this repo, install dependencies, then start the gulp process:

```
	> git clone https://github.com/davidadisanjaya/ReactStarterKit.git
	> cd ReactStarterKit
	> npm install
	> gulp
```

Enjoy. (:
